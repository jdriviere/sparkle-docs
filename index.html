Getting Started

There are many ways to import Sparkle into your development folder, each way being simple and easy to follow.


Install through Bower

This is the favorable way, if you wish to dig deep within the developmental folders of Sparkle and completely customize it. Just follow these steps:

  1. Run the following command.
  2. Link the .css or .min.css file, as well as the .js or .min.js files located in the respective folders, found in your bower_components folder.
  3. Enjoy!
bower install sparkle.css

Install through CDN

You can also install the latest (master) release of the framework. Simply link your stylesheet and script source to the appropriate links.

Note: If you wish to install the latest developmental version of Sparkle, just replace the @[VERSION] with @latest.

<link href="https://maxcdn.bootstrapcdn.com/font-awesome/4.7.0/css/font-awesome.min.css" rel="stylesheet"/>
<link href="https://cdn.jsdelivr.net/gh/jdriviere/sparkle-css@[VERSION]/dist/css/sparkle.min.css" rel="stylesheet">

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.1.1/jquery.min.js"></script>
<script src="https://cdn.jsdelivr.net/gh/jdriviere/sparkle-css@[VERSION]/dist/js/sparkle.min.js"></script>

Note: You must have both properly linked for the framework to function properly.

Install through Git Repo

Go to the following link(s) and download the files you wish to use. It is also an alternative way of ensuring to have the latest developmental version of Sparkle. Or you can simply surf around and download any other version of Sparkle that is available.

Note: This one is especially recommended, if you intend to contribute to this project.

https://github.com/jdriviere/sparkle-css/dist/css/sparkle.min.css

https://github.com/jdriviere/sparkle-css/dist/js/sparkle.min.js
