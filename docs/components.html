Alerts   New!

Alerts are used to display information on your screen. Depending on the type of information you want to display (i.e.: danger, warning, success, info, etc.), Sparkle allows you to style it accordingly. Simply add a alert class to a <div> tag. Alerts can be normal or outlined.

Normal:
This is a default alert.
This is a primary alert.
This is an info alert.
This is a success alert.
This is a warning alert.
This is a danger alert.
This is a light alert.
This is a dark alert.
<div class="alert [make-$type]">
    // Body of Alert
</div>
Outlined:
This is an outlined primary alert.
This is an outlined info alert.
This is an outlined success alert.
This is an outlined warning alert.
This is an outlined danger alert.
This is an outlined light alert.
This is an outlined dark alert.
<div class="alert make-$type make-outline">
    // Body of Alert
</div>

Blockquotes   New!

Sometimes, you just need to take a section from another source and quote it stylistically. Sparkle fulfills that wish too! Simply add blockquote class to your <blockquote> tag. And, as with many of this framework's features, it also has many different styles. All you have to do is add make-$type to the <blockquote> class list, where $type is either one of the following: primary, info, success, warning, danger, light, or dark.

Normal:
This is a default blockquote.
This is a primary blockquote.
This is an info blockquote.
This is a success blockquote.
This is a warning blockquote.
This is a danger blockquote.
<blockquote class="blockquote [make-$type]">
    // Body of Blockquote
</blockquote>

Buttons   New!

One of the most essential elements in a CSS framework: buttons. Sparkle allows you to style anchor tags, div tags, and buttons by simply putting button in its class. This is also the first step to style a button accordingly (see other styling below).

Normal:
<button class="button [make-$type]">Button</button>
Reversed:
<button class="button make-$type make-reverse">Button</button>
Outlined:
<button class="button make-$type make-outline">Button</button>
Reverse-Outlined:
<button class="button make-$type make-reverse make-outline">Button</button>
Ghost:
<button class="button make-$type make-ghost">Button</button>
Link:
<button class="button make-$type make-link">Button</button>

Tags   New!

You can use tags to allow you to identify or emphasize important information. One of the most efficient ways to create a tag is to add <span> element with a tag class. Tags come in various type (default, primary, info, etc.) and can be either normal, outlined, or reversed.

Normal:
DefaultPrimaryInfoSuccessWarningDangerLightDark
<span class="tag make-$type">
    // Body of Tag
</span>
Outlined:
Primary OutlinedInfo OutlinedSuccess OutlinedWarning OutlinedDanger OutlinedLight OutlinedDark Outlined
<span class="tag make-$type make-outline">
    // Body of Tag
</span>
Reversed:
Primary ReversedInfo ReversedSuccess ReversedWarning ReservedDanger ReservedLight ReservedDark Reversed
<span class="tag make-$type make-reverse">
    // Body of Tag
</span>
